# Sunlight Alarm
This uses your wifi network to poll a public time api every 10 seconds to trigger an 'alarm' at a given `RISE_HOUR` and `RISE_MINUTE`. This project uses a Lolin D1 Mini in conjunction with a micro 9g servo.
